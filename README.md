# ✨ rag-chatbot ✨

A cute little Retrieval-Augmented Generation (RAG) chatbot powered by OpenAI 🤍  
Built with an in-memory vector store for fast and lightweight semantic search.

---

## 🌸 Features

- 💬 Chat with your documents
- 🧠 OpenAI-powered responses
- 🔎 Semantic search using vector embeddings
- ⚡ Lightweight in-memory vector database

---

## 🛠️ Tech Stack

- :contentReference[oaicite:0]{index=0} API
- Python
- Vector embeddings
- In-memory retrieval system

---

## 💖 What is RAG?

RAG stands for **Retrieval-Augmented Generation**.

Instead of relying only on AI memory, the chatbot:
1. Searches relevant information from stored documents
2. Retrieves the most relevant context
3. Uses OpenAI to generate smarter and more accurate answers ✨
